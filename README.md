# Vue3 + Node.js 全栈企业后台管理系统

一套完整的前后端分离中后台管理系统，支持权限控制、CRUD、数据可视化、Excel 导出等企业级功能。

## 项目介绍
本项目是一套企业后台管理系统，采用 Vue3 + TypeScript + Node.js + MySQL 全栈开发，涵盖：
- 用户管理 / 角色权限 / 商品管理
- 数据可视化图表
- 操作日志审计
- 通用 CURD、状态切换
- Excel 导入导出
- JWT 登录鉴权


## 技术栈
### 前端(front-end)
- Vue3 + Composition API
- TypeScript
- Element Plus
- Pinia
- Vue Router
- Axios
- Vite

### 后端(back-end)
- Node.js
- Express
- MySQL
- Prisma ORM
- JWT 鉴权
- bcrypt 密码加密

## 核心功能
- 登录 / 注销 / 密码修改（安全加密）
- 权限控制（菜单、路由、接口三级权限）
- 用户管理（增删改查、角色分配）
- 商品管理（状态切换、批量操作）
- 数据图表 ECharts
- 表格数据导出 Excel
- 操作日志记录
- 响应式布局
